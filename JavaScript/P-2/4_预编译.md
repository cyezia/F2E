# 预编译

## 过程

- 1、检查通篇的语法错误

- 1.5、预编译的过程

- 2、解释一行，执行一行

- 函数声明整体提升，变量只有声明提升，赋值是不提升的

## 函数执行之前的步骤

- AO activation object 活跃对象，也叫函数上下文

  1、寻找函数里的形参和变量声明，放到AO里

  2、把实参的参数值赋值给形参

  3、找函数体的函数声明，赋值函数体

  4、执行函数

- GO global object 全局上下文

  1、找变量

  2、找函数声明

  3、执行

- GO === window

- AO GO 为了解决：作用域 作用域链相关所产生的一切问题
