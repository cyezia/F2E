# 字体

- font-size 字体大小

  浏览器设置字体大小时，设置的是高度

- font-weight 字体粗细

  font-weight="lighter|normal|bold|bolder|100-900"

- font style 字体风格

  font-style="italic|oblique" 斜体

- font family 设置字体

  通用字体：arial

  可设置多个值：font-family:" Times New Roman ", Georgia, serif;

  有空格的字体标识、中文字体标识需要用" "

- color 字体颜色

  英文单词颜色设置

  十六进制的颜色值（#00ff00）

  rgn(0, 255, 0)
  
- 光学的三原色：red green blue

  对应十六进制：00-ff 00-ff 00-ff

  对应rgb值：0-255 0-255 0-255
