# 节点属性

- 元素节点 -> 1

- 属性节点 -> 2

- 文本节点 -> 3

- 注释节点 -> 8

- document -> 9

- DocumentFragment -> 11

## nodeName 节点名

- 元素节点的nodeName是大写的，只读

- toLowerCase() 变小写

- toUpperCase() 变大写

## nodeValue 节点值

- 可写 属性、注释、文本可用

## getAttributeNode 获取属性节点

## nodeType 节点类型

## attributes 是一个元素的属性集合
