# CSS书写顺序

## 顺序如下

- 显示属性：

  display
  
  position
  
  float
  
  clear

- 自身属性：

  width
  
  height
  
  margin
  
  padding
  
  border
  
  background

- 文本属性：

  color
  
  font
  
  text-align
  
  vertical-aligin
  
  whitespace

## font 简写

- font-style

- font-weight

- font-size

- line-height

- font-family

- 按以上顺序写，举例：font: italic bold 12px/20px "微软雅黑"
